# How To Run? clone this project & run
```bash
git clone https://github.com/leekHotline/cortex-react.git
cd cortex-react
pnpm dev
```

# 安装必须的依赖包 please install package
## 通过脚手架vite构建项目 拥有丰富的vue,react,ts模板
```bash
pnpm create vite@latest project-name
```

## biome取代eslint
```bash
pnpm add -D @biomejs/biome
```

## 添加现代化的ui和动画效果 (framer-motion专为react设计) (gasp专为原生js设计)
```bash
pnpm add framer-motion 
pnpm add gsap 
```
